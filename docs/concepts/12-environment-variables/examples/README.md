# Environment Variables Examples

This directory contains practical code examples demonstrating environment variables in Next.js.

## Examples

### 1. Basic Environment Variables (`01-basic-env.tsx`)
Demonstrates environment variable usage:
- Server variables
- Client variables (NEXT_PUBLIC_)
- Type-safe configuration

## How to Use

1. Create .env.local file
2. Add your variables
3. Use in code
4. Test access

## Key Concepts

- Server variables: No prefix, server-only
- Client variables: NEXT_PUBLIC_ prefix
- Never expose secrets
- Validate required variables
- Use type-safe config
