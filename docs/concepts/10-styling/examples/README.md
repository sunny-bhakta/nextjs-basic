# Styling Examples

This directory contains practical code examples demonstrating Next.js styling approaches.

## Examples

### 1. Tailwind CSS (`01-tailwind-css.tsx`)
Demonstrates Tailwind CSS usage:
- Basic utility classes
- Responsive design
- Component styling
- Dark mode

### 2. CSS Modules (`02-css-modules.tsx`)
Shows CSS Modules usage:
- Scoped CSS classes
- Multiple classes
- Conditional classes
- Component styling

## How to Use

1. **Copy the code** from example files
2. **Create corresponding files** in your `app/` directory
3. **For CSS Modules**: Create `.module.css` files
4. **For Tailwind**: Ensure Tailwind is configured
5. **Run your app**: `npm run dev`
6. **Test styles**: Verify styling works

## File Structure

```
app/
├── components/
│   ├── Button.tsx
│   ├── Button.module.css (for CSS Modules)
│   └── ...
├── globals.css (for Tailwind)
└── page.tsx
```

## Key Concepts

- Tailwind: Utility-first CSS framework
- CSS Modules: Scoped CSS with .module.css
- Choose based on project needs
- Both work great with Next.js
