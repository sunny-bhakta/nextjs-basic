## Overview

The Pages Router is the original routing system in Next.js using the `pages/` directory. While still supported, the App Router is recommended for new projects. Learn this for maintaining legacy projects or understanding Next.js evolution.

## Key Topics

- Directory Structure - Understanding the `pages/` folder
- Custom App (`_app.js`) - Customizing the App component
- Custom Document (`_document.js`) - Customizing the HTML document
- API Routes - Creating API endpoints in `pages/api/`
- Special Files - `404.js`, `500.js`, `_error.js`
