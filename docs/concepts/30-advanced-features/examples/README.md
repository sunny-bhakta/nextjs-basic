# Advanced Features Examples

This directory contains practical code examples demonstrating advanced Next.js features.

## Examples

### 1. Edge Functions (`01-edge-functions.ts`)
Demonstrates Edge Functions:
- Edge runtime configuration
- Edge middleware
- Edge API routes

### 2. Streaming (`02-streaming.tsx`)
Shows streaming:
- Suspense boundaries
- Streaming SSR
- Progressive loading

### 3. Server Components (`03-server-components.tsx`)
Demonstrates Server Components:
- Advanced patterns
- Data fetching
- Component composition

## How to Use

1. **Copy example code**: Adapt to your use case
2. **Configure runtime**: Set edge or nodejs
3. **Test streaming**: Verify Suspense works
4. **Optimize**: Monitor performance

## Key Concepts

### Edge Functions
- Low latency
- Global distribution
- Limited resources

### Streaming
- Progressive rendering
- Suspense boundaries
- Faster perceived performance

### Server Components
- Server-side rendering
- Reduced bundle size
- Direct data access

## Best Practices

1. **Use Edge for simple operations**: Low latency needs
2. **Stream with Suspense**: Faster loads
3. **Leverage Server Components**: Reduce bundle
4. **Monitor performance**: Track metrics
5. **Test edge cases**: Different scenarios
