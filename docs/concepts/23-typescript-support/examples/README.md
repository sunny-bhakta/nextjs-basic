# TypeScript Support Examples

This directory contains practical code examples demonstrating TypeScript usage in Next.js.

## Examples

### 1. Basic Types (`01-basic-types.tsx`)
Demonstrates basic TypeScript:
- Interfaces and types
- Component props
- Type definitions

### 2. API Routes Types (`02-api-routes-types.ts`)
Shows type-safe API routes:
- Request/response types
- Route handler types
- Error handling types

### 3. Component Types (`03-component-types.tsx`)
Demonstrates component typing:
- Props interfaces
- Generic components
- Type inference

## How to Use

1. **Copy the code** from any example file
2. **Create the corresponding files** in your `app/` directory
3. **Ensure TypeScript is configured**: Check `tsconfig.json`
4. **Run your Next.js app**: `npm run dev`
5. **Check types**: Use TypeScript compiler

## Key Concepts

### Type Safety
- Catch errors at compile time
- Better IDE support
- Safer refactoring

### Type Definitions
- Interfaces for objects
- Types for unions
- Generics for reusability

### Next.js Types
- Built-in type definitions
- Route parameter types
- API route types

## Best Practices

1. **Enable strict mode**: Maximum type safety
2. **Type everything**: Avoid `any`
3. **Use interfaces**: For object shapes
4. **Type props**: All component props
5. **Type API routes**: Request/response types
