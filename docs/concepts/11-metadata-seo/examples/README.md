# Metadata & SEO Examples

This directory contains practical code examples demonstrating Next.js metadata and SEO.

## Examples

### 1. Static Metadata (`01-static-metadata.tsx`)
Demonstrates static metadata:
- Basic metadata
- Metadata with Open Graph
- Complete metadata
- Metadata templates

### 2. Dynamic Metadata (`02-dynamic-metadata.tsx`)
Shows dynamic metadata generation:
- Metadata from route params
- Complete dynamic metadata
- Error handling

## How to Use

1. Copy code from example files
2. Create corresponding files in your app
3. Test metadata in browser dev tools
4. Verify SEO tags

## Key Concepts

- Use Metadata type for type safety
- Export metadata object
- Use generateMetadata for dynamic
- Include Open Graph for social sharing
- Test with SEO tools
