# File System Examples

This directory contains practical examples demonstrating file system usage in Next.js.

## Examples

### 1. Public Folder (`01-public-folder.md`)
Demonstrates public folder:
- Folder structure
- Static file serving
- Asset organization

### 2. Static Assets (`02-static-assets.tsx`)
Shows static asset usage:
- Image components
- Font loading
- Icon usage

### 3. File Serving (`03-file-serving.ts`)
Demonstrates file serving:
- Static file serving
- Dynamic file paths
- Asset optimization

## How to Use

1. **Review structure**: Understand public folder
2. **Organize assets**: Create logical structure
3. **Use examples**: Follow patterns
4. **Optimize assets**: Compress files

## Key Concepts

### Public Folder
- Served at root
- Static files
- No processing

### Static Assets
- Images, fonts, icons
- Optimized serving
- CDN support

### File Serving
- Automatic serving
- Cache headers
- Performance

## Best Practices

1. **Organize logically**: Clear folder structure
2. **Optimize files**: Compress assets
3. **Use Next.js Image**: For images
4. **Version assets**: Cache busting
5. **Use CDN**: For large files
