# Redirects & Rewrites Examples

This directory contains practical code examples demonstrating redirects and rewrites in Next.js.

## Examples

### 1. Redirects (`01-redirects.ts`)
Demonstrates redirects:
- Simple redirects
- Pattern redirects
- Dynamic redirects in middleware

### 2. Rewrites (`02-rewrites.ts`)
Shows rewrites:
- API proxying
- Pattern rewrites
- Internal rewrites
- Dynamic rewrites

## How to Use

1. Add to next.config.js
2. Or use middleware
3. Test redirects/rewrites
4. Verify behavior

## Key Concepts

- Redirects change URL
- Rewrites keep URL same
- Use permanent: true for 301
- Pattern matching with :path*
- Test all redirects/rewrites
