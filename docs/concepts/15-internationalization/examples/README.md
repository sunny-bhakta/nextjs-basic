# Internationalization Examples

This directory contains practical code examples demonstrating internationalization in Next.js.

## Examples

### 1. next-intl Setup (`01-next-intl.tsx`)
Demonstrates next-intl:
- Configuration
- Locale routing
- Client and server components
- Language switcher

## How to Use

1. Install next-intl
2. Set up configuration
3. Create translation files
4. Use translations
5. Test all locales

## Key Concepts

- Use next-intl for i18n
- Set up locale routing
- getTranslations for Server Components
- useTranslations for Client Components
- Organize translation files
