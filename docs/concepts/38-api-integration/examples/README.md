# API Integration Examples

This directory contains practical code examples demonstrating API integration in Next.js.

## Examples

### 1. REST API (`01-rest-api.ts`)
Demonstrates REST API integration:
- API client setup
- Request/response handling
- Error handling

### 2. GraphQL (`02-graphql.ts`)
Shows GraphQL integration:
- Apollo Client setup
- Queries and mutations
- Server-side rendering

### 3. tRPC (`03-trpc.ts`)
Demonstrates tRPC integration:
- tRPC setup
- Type-safe calls
- Server/client integration

## How to Use

1. **Install dependencies**: Required packages
2. **Set up API client**: Configure base URL
3. **Copy example code**: Adapt to your API
4. **Test integration**: Verify API calls

### REST API

1. **Use API clients**: Centralized logic
2. **Handle errors**: Proper error handling
