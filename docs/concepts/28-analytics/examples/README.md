# Analytics Examples

This directory contains practical code examples demonstrating analytics integration in Next.js.

## Examples

### 1. Vercel Analytics (`01-vercel-analytics.tsx`)
Demonstrates Vercel Analytics:
- Basic setup
- Custom events
- Speed insights

### 2. Google Analytics (`02-google-analytics.tsx`)
Shows Google Analytics:
- GA4 setup
- Event tracking
- Page view tracking

### 3. Web Vitals (`03-web-vitals.tsx`)
Demonstrates Web Vitals:
- Core Web Vitals tracking
- Performance monitoring
- Custom reporting

## How to Use

1. **Install dependencies**:
   ```bash
   npm install @vercel/analytics web-vitals
   ```

2. **Set up analytics**: Follow example setup

3. **Configure tracking**: Add your IDs

4. **Test events**: Verify tracking works
